## 圖片圖表嵌入問題  
* 轉檔時出現不知名的問題，後來發現：只要**不加入圖片**，就可以轉檔。只好暫時放棄圖片！
    * 可以加入**60KB以下**的圖片！超過60KB的無法加入！
  
* HTML圖表，只可嵌入2個：  
    * 其它的嵌入就會失敗，不知何因？  
1. 四界的十二種特相或性質.html  
2. 四大（界）差別觀的：3a-Four Elements12122004.html
